# WL: WaggleLab

WaggleLab is a Python library for testing Active Learning algorithims. 

## Installation
Use the package manager pip to install WL
```bash
pip install wagglelab
```


## Requirements
```python
tensorflow==2.3.0
```

## Usage
```python
import wagglelab as wl
```
